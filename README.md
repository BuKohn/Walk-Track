# Walk&Track
## Walk and Track your progress
## Установка

### Вариант 1. Установить готовый APK

1. Скачать APK из [Releases](https://github.com/BuKohn/Walk-Track/releases)
2. Установить на Android-устройство
3. Разрешить установку из неизвестных источников

### Вариант 2. Собрать из исходного кода

1. Клонировать репозиторий
2. Ввести в командной строке в корне проекта:
```bash
.\gradlew.bat assembleDebug
```
3. APK файл появится в: app/build/outputs/apk/debug/app-debug.apk
